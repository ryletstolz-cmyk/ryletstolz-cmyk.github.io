    <h1>Hytale downloader CLI tool</h1>
    You can interface with this tool by entering your docker terminal. More information here.

    You can use the following commands:

   <ul>
  <li>hytale-downloader :----&gt;Download the latest release</li>
  <li>hytale-downloader -print-version :----&gt;	Show game version without downloading</li>
  <li>hytale-downloader -version :----&gt;Show hytale-downloader version</li>
   <li>hytale-downloader -check-update :----&gt;Check for hytale-downloader updates</li>
   <li>hytale-downloader -patchline pre-release :----&gt;Download from pre-release channel</li>
   <li>hytale-downloader -skip-update-check	:----&gt;Skip automatic update </li>
  
</ul> 

<h2>Updating Hytale Server</h2>
<h3>If you want to automatically update Hytale to the newest version:

Enter your container terminal
Run hytale-downloader to download the latest version<br>
Restart the container with docker compose restart<br>
The server will automatically detect and extract the update<br>
The update process only replaces server binaries - your configs, worlds, and mods are preserved.</h3>
